# IF · 思维提取方法论

> 「IF he were here, what would he do?」

## 什么是 IF

IF 不是收集一个人说过的话。IF 是挖出他的**认知操作系统**——他怎么看世界、怎么做判断、怎么从一个问题走到一个结论。

提取结果是一套可以**直接运行的 AI 人格**，你可以跟它对话、让它帮你分析问题、用它的眼睛看你的困境。

## 核心理念

一个人的思维可以拆成三层：

```
表层：表达方式（HOW they talk）
 ↓
中层：思维框架（HOW they think）
 ↓
底层：认知操作系统（WHY they think that way）
```

IF 穿透三层——不只是模仿语气，是**逆向推导他的思考流程**。

## 五阶段提取流程

### 阶段1：全景扫描
建立认知地图：统计素材、主题聚类、绘制时间线、标注演化节点。

### 阶段2：模式发现
从素材中发现候选思维模式：
- 人工密集阅读（小样本，高质量）
- 计算辅助（大样本，词频分析/主题建模）
- 结构模式提取（论证结构标注）

### 阶段3：框架筛选
三重验证法筛选「真模型」：
| 验证 | 问题 |
|------|------|
| 跨域复现 | 同一框架出现在 ≥2 个不同领域？ |
| 生成力 | 用这个模型能推断他对新问题的立场？ |
| 排他性 | 不是所有聪明人都这样想？ |

三重全过 → 心智模型。过1-2重 → 决策启发式。0重 → 丢弃。

### 阶段4：操作系统逆向 ⭐
**这是 IF 的护城河。** 反向推导他的：
- **选题策略**：他怎么决定「聊什么」
- **论证结构**：他怎么从「一个想法」走到「一个结论」
- **类比生成机制**：他怎么找到那个让人秒懂的比喻
- **反直觉推导链**：他怎么看到常识的反面

### 阶段5：人格构建
组装所有提取结果为可运行的 AI 人格，验证质量。

## 与女娲（Nuwa）的分工

IF 和 [女娲](../autonomous-ai-agents/huashu-nuwa/SKILL.md) 互补：

| 你要什么 | 用哪个 | 产出 |
|---------|--------|------|
| 「用鹤老师的脑子跟我对话」 | 女娲 | 可对话的 AI 人格 Skill |
| 「鹤老师是怎么推导出这些结论的」 | IF | 认知提取报告 + 操作系统分析 |

**经验法则**：女娲做「表层+中层」（表达DNA+心智模型），IF 做「底层」（操作系统逆向）。两者可以组合使用——先女娲出 Skill，再 IF 出方法论报告。

## 适用边界

- ✅ 公开人物（≥100篇文字素材）
- ✅ 身边熟人（有聊天记录，参考 SoulCraft/CAPS 框架增强）
- ✅ 自己（需警惕自我认知偏差）
- ❌ 素材 <30篇 → 只能快速画像
- ❌ 大量代笔/公关稿 → 无法区分真观点

## 常见陷阱

| 陷阱 | 避免方式 |
|------|---------|
| 金句崇拜 | 看重复次数，不看单次表达 |
| 平滑化矛盾 | 矛盾本身就是特征 |
| 过度抽象 | 排他性验证——不只你聪明 |
