# 思维提取方法论 v1.0

> 从一个人的文字/语音/视频内容中，系统性地提取其认知操作系统——不是「他说了什么」，而是「他怎么想」。

---

## 一、核心概念定义

### 1.1 提取目标层的三层结构

一个人的思维可以被拆成三层，从浅到深：

```
表层：表达方式（HOW they talk）
 ↓
中层：思维框架（HOW they think）
 ↓
底层：认知操作系统（WHY they think that way）
```

| 层级 | 内容 | 可迁移性 | 举例（鹤老师） |
|------|------|---------|-------------|
| **表层 · 表达** | 句式、词汇、节奏、幽默方式 | 低——换了话题就失效 | 「通俗比准确重要一万倍」→ 短句+反问+断言 |
| **中层 · 框架** | 心智模型、决策启发式、论证结构 | 中——同类问题可复用 | 「元规则」→ 任何事先找实际规则 |
| **底层 · 操作系统** | 选题策略、类比生成机制、反直觉推导链 | 高——可用于任何新领域 | 他看「长租公寓」和看「瑞士金鱼法」用的是同一个推导引擎 |

### 1.2 提取 ≠ 复制

提取的结果不是「此人的原话拼凑」，而是：
- **心智模型**：他反复使用的那几个看世界的镜片
- **决策启发式**：他做判断时自动触发的快速规则
- **操作手册**：他分析问题、生成洞察、构建表达的具体步骤
- **盲区和张力**：他也没想清楚的地方

## 二、输入要求

### 2.1 素材类型与质量等级

| 素材类型 | 揭示什么 | 质量等级 |
|---------|---------|---------|
| **长文/Essay**（3000字以上系统论述） | 思维结构、论证链、框架体系 | ⭐⭐⭐⭐⭐ |
| **短文章/博客**（500-2000字） | 观点浓缩、表达DNA、类比能力 | ⭐⭐⭐⭐ |
| **长访谈/播客**（1小时以上即兴对话） | 即兴反应、被追问时的处理、立场变化 | ⭐⭐⭐⭐ |
| **社交媒体碎片**（微博/Twitter/即刻） | 即时反应、情绪倾向、高频词汇 | ⭐⭐⭐ |
| **视频/音频** | 语气、停顿、非语言信息 | ⭐⭐⭐ |
| **决策记录**（内部备忘录、邮件） | 真实决策逻辑 vs 公开表达 | ⭐⭐⭐⭐⭐ |
| **二手转述/他人评价** | 外部视角、盲点 | ⭐⭐ |

### 2.2 最低素材门槛

| 提取深度 | 最低素材量 | 预期质量 |
|---------|-----------|---------|
| **快速画像** | 30篇短文 / 3小时访谈 | 60-70%还原度 |
| **标准提取** | 100篇短文 / 10小时素材 | 80-85%还原度 |
| **深度提取** | 300篇+文章 / 30小时素材，覆盖≥2年跨度 | 90%+还原度 |

**关键约束**：素材必须覆盖 **≥2个不同时期**，才能分辨「偶发观点」和「稳定框架」。时间跨度越大，提取质量越高。

### 2.3 素材清洗

- 删除纯元数据（标题、日期、标签、推广链接）
- 删除重复内容（同一文章的多格式版本取最好的那个）
- 标注素材时间戳，按时间线排序
- 区分「免费公开内容」和「付费/内部内容」（付费内容可能有刻意包装）

## 三、提取流程（五阶段）

### 阶段1：全景扫描（Surface Mapping）

**目标**：建立对素材的整体认知地图，不做深度分析。

**操作**：
1. **统计基础数据**：总篇数、时间跨度、字数/时长分布、主题分类
2. **时间线绘制**：按时间排序所有素材，标记关键节点（出书、跳槽、品牌改名等）
3. **主题聚类**：用无监督方法或人工标记，将所有素材归入3-8个主题维度
4. **演化标注**：对比早期和最近的素材，标记「哪些变了、哪些没变」

**输出**：素材全景图 + 主题分布 + 演化趋势标注

**质量检查**：
- [ ] 能说出这个人在什么时期讨论什么话题？
- [ ] 能标记出≥3个内容演化节点？
- [ ] 能区分「早期观点」和「近期观点」？

---

### 阶段2：模式发现（Pattern Discovery）

**目标**：从素材中自动/半自动地发现候选思维模式。

**方法A：人工密集阅读（小样本，高质量）**
- 随机抽样30-50篇，逐篇标记：核心论点、论证结构、类比类型、结论句式
- 寻找≥3次出现的模式 → 标记为「候选心智模型」

**方法B：计算辅助（大样本，高效率）**
- 词频分析：高频实词、高频句式、禁忌词
- 主题建模（LDA/Topic Modeling）：自动发现隐藏主题
- 共现网络分析：哪些概念经常一起出现
- 情感时间序列：情感倾向随时间的演化

**方法C：结构模式提取**
- 对所有素材做**论证结构标注**：每个素材的「提出问题 → 展开分析 → 得出结论」路径
- 寻找**稳定结构**：如果80%的文章遵循相同结构 → 这是他的操作习惯

**输出**：候选心智模型列表（通常15-30个） + 候选结构模式

**质量检查**：
- [ ] 每个候选模型有≥3条原文证据？
- [ ] 不同时期、不同话题都出现？→ 跨域复现
- [ ] 标注了每条证据的来源和时间？

---

### 阶段3：框架筛选（Framework Filtering）

**目标**：从候选模型中筛选出「真模型」（稳定框架），剔除「随口一说」（偶发观点）。

**三重验证法**：

| 验证 | 问题 | 通过标准 |
|------|------|---------|
| **跨域复现** | 同一个思维框架出现在≥2个不同领域/话题？ | 出现在≥2个不同领域 |
| **生成力** | 用这个模型能推断此人面对新问题的立场？ | 能推出≥1个可验证的推论 |
| **排他性** | 不是所有聪明人都会这样想？这个模型体现了独特性？ | 能找到≥1个「不这么想也很合理」的反例 |

**筛选规则**：
- 三重全过 → **心智模型**（核心产出）
- 过1-2重 → 降级为**决策启发式**（次产出）
- 0重 → 丢弃

**数量控制**：
- 心智模型：3-7个（宁少勿多，3个深刻的远好于10个浅薄的）
- 决策启发式：5-10条
- 按排他性强度排序（越独特越靠前）

**输出**：最终心智模型列表 + 决策启发式列表

---

### 阶段4：操作系统逆向（OS Reconstruction）⭐

**目标**：从心智模型中反向推导出他的**认知操作流程**——他怎么从「看到一个问题」走到「给出一个结论」。

**操作**：
1. **选题策略提取**：他选择讨论一个话题时，有什么规律？
2. **论证结构提取**：标记20篇代表文章的论证流程，寻找共同路径
3. **类比生成机制**：提取所有类比，分类来源，分析映射关系
4. **反直觉推导链**：回溯反常识结论的推导过程，找到触发条件

**输出**：认知操作手册（选题策略 + 论证模板 + 类比生成规则 + 反直觉触发条件）

**这是整个提取方法论的护城河。** 前三个阶段很多框架都能做，但阶段4是从「知道他想什么」到「能像他一样思考」的关键一步。

---

### 阶段5：人格构建与验证（Persona Construction）

**目标**：将所有提取结果组装为一个可运行的 AI 人格。

**验证方法**：

1. **已知测试**：选3个此人公开表态过的问题 → AI 用此人的思维框架回答 → 对比实际立场
2. **边缘测试**：选1个此人从未讨论过的问题 → AI 推断 → 检查是否合理且不越界
3. **风格盲测**：混入此人的真实文字和 AI 生成的文字 → 看能否区分

**通过标准**：
- [ ] 每个心智模型有≥2个不同领域的证据
- [ ] 3-7个模型，每个有应用场景和失效条件
- [ ] 表达DNA有辨识度（读100字能认出是谁）
- [ ] 写了≥3条诚实边界
- [ ] 保留了≥2对内在矛盾
- [ ] 一手素材占比 >50%

## 四、方法论适用边界

### 4.1 适用场景
- ✅ 公开人物（作家、思想家、企业家、内容创作者）
- ✅ 身边熟人（有聊天记录，参考 SoulCraft/CAPS 框架增强）
- ✅ 历史人物（注意传记偏差，多源交叉验证）
- ✅ 自己（需警惕自我认知偏差）
- ❌ 素材 <30篇 → 只能快速画像
- ❌ 大量代笔/公关稿 → 无法区分真观点

### 4.2 常见陷阱

| 陷阱 | 表现 | 避免方式 |
|------|------|---------|
| **金句崇拜** | 把说得漂亮的话当成核心信念 | 看行动，看重复次数 |
| **平滑化** | 忽略矛盾，把观点修得太一致 | 矛盾本身就是特征 |
| **过度抽象** | 把每个人的框架都提取成相同的通用道理 | 排他性验证 |
| **时效性忽略** | 把10年前的观点当成当前框架 | 标注时间，区分早期/近期 |
| **素材质量忽视** | 把软文/广告/代笔当成一手来源 | 区分素材类型 |
